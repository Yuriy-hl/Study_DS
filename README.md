# Практическое изучение Data Science

Этот репозиторий представляет собой мой личный конспект-практикум по книге Джоэла Граса (Joel Grus) "Data Science: наука о данных с нуля" 

Здесь я шаг за шагом реализую алгоритмы и концепции, разобранные в книге, проверяю их на практике и дополняю комментариями. 

Цель - глубоко понять базовые принципы и математику, лежащие в основе науки о данных, используя базовый python и numpy

Этот проект будет пополняться по мере прохождения глав
